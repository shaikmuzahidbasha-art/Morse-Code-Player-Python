# Morse Code Player with Text-to-Speech & Audio Generation

A Python GUI application that converts Text ↔ Morse Code, plays Morse audio in real-time, generates downloadable WAV files, and supports Text-to-Speech.

## Features

- Text to Morse Conversion
- Morse to Text Conversion
- Real-Time Morse Audio Playback
- Download Morse Audio as WAV
- Text-to-Speech
- Copy Morse Output

## Technologies Used

- Python
- Tkinter
- NumPy
- gTTS
- playsound
- wave
- winsound
- pyperclip
